Arbiter

Automated, multi-agent code review for every pull request.

Arbiter is a GitHub App that reviews each pull request the moment it opens. Instead of one generalist model, it runs three specialist AI agents in parallel — code quality, test coverage, and documentation — then a supervisor synthesizes their notes into a single verdict posted right on the PR.

code quality test coverage documentation
→ supervisor → ✅ PR comment
